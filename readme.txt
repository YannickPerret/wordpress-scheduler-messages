== Description ==
Plugin pour afficher des messages personnalisés entre des périodes définies.
- Utilise des shortcodes et des widgets.
- Gestion des périodes avec une interface d'administration.
- Options pour personnaliser les styles (couleurs, bordures, transparence).

== Installation ==
1. Téléchargez le plugin depuis WordPress.org.
2. Décompressez le fichier zip dans le répertoire `/wp-content/plugins/`.
3. Activez le plugin via le menu **Extensions** de WordPress.
